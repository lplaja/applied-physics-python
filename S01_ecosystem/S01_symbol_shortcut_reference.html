Units (for f-strings & labels)

SymbolMeaningUsage in f-string
Åangstromf"{d:.1f} Å"
°degreef"{θ:.1f}°"
µmicrof"{w₀:.1f} µm"
In Python code — use Greek letters as variable names:
λ = 800e-9   θ = λ / (π * w₀)

In f-strings & comments — use all symbols freely:
print(f"Δν = {dnu:.2e} Hz")
